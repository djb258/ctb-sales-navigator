Quote Presentation

All quoted costs per insurance line, carrier comparison, close meeting.

  • Summary cards: company, employees, current premium, quotes received
  • Quotes grouped by benefit type (medical, dental, vision, life)
  • Per-carrier rates: EE, ES, EC, FAM tiers + annual cost
  • Total cost comparison: carrier totals vs current premium with delta

Data: sales.sales_quotes + sales.sales_factfinder (for current premium baseline)

On Close: Promote to Client

When the sale closes, the sales_id promotes to client_id via the CL sovereign.

  • CL sovereign ID joins sales → client
  • Process 800 (Client Mint) mints new client_id
  • Fact Finder data seeds the client record
  • Selected quotes become the initial plan setup

Contract: sales/contracts/promote_to_client.json
